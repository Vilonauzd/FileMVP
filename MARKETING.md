# FileMVP

**FileMVP** is a simple iOS utility for creating and renaming files with full control over file names and extensions.

iOS limits how users can create and rename files. FileMVP removes unnecessary friction by offering a predictable, straightforward way to work with plain text files.

---

## What FileMVP Does

- Create new files instantly
- Rename files freely, including file extensions
- Works entirely on-device
- No accounts, no ads, no tracking

FileMVP focuses on doing one thing clearly and reliably.
